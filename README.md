Первый полный и качественный русификатор для **Hytale**, разработанный профессиональной студией **RemakeStudio**. Мы перевели всё: от интерфейса и системных сообщений до шрифтов и серверных листов, обеспечив полную совместимость с архитектурой игры.

## Установка

1. Скачайте `HytaleRussifier.exe` из раздела [Releases](../../releases)
2. Запустите установщик предварительно закрыв игру
3. Выберите путь к игре (папка Hytale)
4. Нажмите "Установить русификатор"
5. Готово! Запустите игру

## Важно

⚠️ **После каждого обновления Hytale через лаунчер игра будет выдавать ошибку из-за измененных файлов. Придется переустановить игру заново и применить русификатор снова.**

## Ручная установка

Если установщик не работает:

1. Распакуйте `Assets.zip` в папке `Hytale\install\release\package\game\latest` (должен получиться путь `Hytale\install\release\package\game\latest\Assets`)
2. Удалите архив `Assets.zip`
3. Скопируйте папку `install` из этого репозитория в корень игры с заменой
4. Измените язык в настройках игры при необходимости
